# roman-numerals-encoder

Coding kata challenge for encoding decimal numbers up to 3999 as Roman numerals (standard notation)

## Setup

- Install dependencies: `npm install`

## Usage

- Type-check only: `npm run typecheck`
- Compile to `dist/`: `npm run build`
- Compile and run in node (CLI): `npm start` (or `npm start:node`)
- Compile and run in browser: `npm start:browser`

## Unit tests

- Run tests once: `npm test`
- Run tests in watch mode: `npm run test:watch`
